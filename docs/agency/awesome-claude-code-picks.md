# Awesome Claude Code — Picks for Nexus Agency

Curated from [awesome-claude-code](https://github.com/hesreallyhim/awesome-claude-code). Only tools that matter for our stack.

## Install Now (High Impact)

| Tool | What | Why We Need It |
|------|------|---------------|
| [Claude Squad](https://github.com/smtg-ai/claude-squad) | Manage multiple Claude agents in separate workspaces | Run agents for ReadyRide AND ReadyCar simultaneously from one terminal |
| [ccflare](https://github.com/snipeship/ccflare) | Web-based usage dashboard | Track how much API we're burning per client |
| [Claude Hub](https://github.com/claude-did-this/claude-hub) | Connect Claude to GitHub via webhooks | Auto-review PRs, auto-respond to issues |
| [cchooks](https://github.com/GowayLee/cchooks) | Python SDK for Claude Code hooks | Automate pre/post actions on every tool call |
| [Trail of Bits Security Skills](https://github.com/trailofbits/skills) | Code auditing with CodeQL/Semgrep | Catch vulnerabilities before production |
