## Design Notes for Screenshot Creation

- **Background:** Use Slack's dark mode (#1A1D21) for modern look
- **Accent colors:** Green (#36C5F0 for Slack blue, #2EB67D for green status)
- **Font:** Slack uses Lato — use system sans-serif if recreating
- **Buttons:** Slack-style outlined buttons with hover state
- **Create in:** Real Slack workspace (preferred) or Canva mockup template
- **Resolution:** Export at 2x (retina) for crisp screenshots in deck
- **File format:** PNG with transparent background for slides, JPG for web
